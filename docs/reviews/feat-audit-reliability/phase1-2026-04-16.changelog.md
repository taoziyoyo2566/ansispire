## Intent
To transform the audit relay from a "best-effort" poller into a "zero-data-loss" infrastructure component, ensuring that the Ansispire control system maintains a complete and reliable audit trail.

## Explicitly NOT Done
- Real-time push (webhooks) from Semaphore (kept as a future option).
- Integration with external log aggregators (ELK/Splunk).

### 2. Functional Proof (Logic Verification)
The refactored `tick` loop now handles pagination:
```python
while page <= MAX_PAGES:
    events = fetch_page(opener, page)
    # ... logic to find cursor ...
    if reached_cursor: break
    page += 1
```
This ensures that even if the relay is offline for 1000 events, it will fetch all 20 pages (at limit=50) until it hits the last known timestamp.
