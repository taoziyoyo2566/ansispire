### 2. 事实准确性——Prompt Caching 认知错误（严重）

**IVG 声称**：应在 `CLAUDE.md` 中将 `SUMMARY.md` 放入 Cache，`investigations/` 设为按需加载。

**实际情况**：

- Prompt Caching 是 **Anthropic API 层面**的特性，通过在 API 请求的 `system`/`user` 消息里附加 `"cache_control": {"type": "ephemeral"}` header 来标记缓存断点。
- 它由**调用方代码**控制，与 `CLAUDE.md` 文本内容完全无关。
- 在 **Claude Code CLI** 场景下，用户无法通过 `CLAUDE.md` 指令控制哪些内容被 API 缓存——Claude Code 内部自行管理 API 调用。
- `CLAUDE.md` 里的「分层加载」（先读 SUMMARY.md，再读 TODO.md）是**指导 AI 按顺序读取哪些文件**，两者机制完全不同。

**影响**：IVG §6 行动方案（「基于 Prompt Caching 重写 CLAUDE.md」）方向错误，若执行会产生无效改动。
